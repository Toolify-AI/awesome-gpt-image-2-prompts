<a id="project-context"></a>
## プロジェクト背景

2026-04-17 時点で、`GPT Image 2` という名称の OpenAI 公式な単独プロダクトページや API ドキュメントは確認できませんでした。したがって、このリポジトリでの呼称は Toolify と公開された SNS 議論に基づく便宜的なラベルです。

**推論メモ:** それでもこのデータには実用的な価値があります。公開の場でこの画像系スタックを使った、試した、あるいは議論したと見られるクリエイターの Prompt を集約し、検索しやすい研究用リポジトリとして整理しました。

<a id="observed-timeline"></a>
## 観測タイムライン

- `2025-03-25` OpenAI は ChatGPT と API 向けの GPT-4o 画像生成を発表しました。
- `2026-04-04` 複数の X 投稿で、`maskingtape-alpha`、`gaffertape-alpha`、`packingtape-alpha` などの名前で Arena 型テストに現れた新しい画像モデルらしきものが話題になりました。
- `2026-04-17` Toolify には `GPT Image 2` の Prompt ページが公開されていますが、確認できた OpenAI 公式の画像ドキュメントでは `gpt-image-1.5`、`gpt-image-1`、`gpt-image-1-mini` と GPT-4o 画像生成が公開ベースです。
