<a id="project-context"></a>
## 專案背景

截至 2026-04-17，我們沒有查到 OpenAI 官方單獨以 `GPT Image 2` 命名的產品頁或 API 文件。因此倉庫中的這個名稱，更接近 Toolify 與社群討論裡的社群叫法，而不是對官方命名的確認。

**推論說明:** 不過這批資料仍然很有價值，因為其中收錄了大量疑似已獲得權限、已參與測試、或已公開討論過這套圖像能力的創作者 Prompt，我們把它整理成了可搜尋的研究倉庫。

<a id="observed-timeline"></a>
## 觀察時間線

- `2025-03-25` OpenAI 發布了 GPT-4o 圖像生成功能，覆蓋 ChatGPT 與 API。
- `2026-04-04` 多則 X 貼文開始討論一個疑似新圖像模型在 Arena 式測試中出現，代號包含 `maskingtape-alpha`、`gaffertape-alpha` 與 `packingtape-alpha`。
- `2026-04-17` Toolify 已公開上線 `GPT Image 2` Prompt 頁面，而我們查到的 OpenAI 官方圖像文件仍主要公開記錄 `gpt-image-1.5`、`gpt-image-1`、`gpt-image-1-mini` 與 GPT-4o 圖像生成。
